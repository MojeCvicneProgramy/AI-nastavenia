== Description ==
The Under Construction plugin allows you to temporarily display a custom maintenance page to visitors while your site is undergoing updates or maintenance. It returns a 503 HTTP status code to inform search engines and users that the site is temporarily unavailable.

== Installation ==
1. Upload the `maintenance-mode` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the settings under the 'Maintenance Mode' menu in the admin dashboard.

== Usage ==
Once activated, you can enable or disable maintenance mode from the plugin settings. When enabled, non-logged-in users will see the maintenance page, while logged-in users with appropriate permissions will have full access to the site.

== Features ==
- Customizable maintenance message.
- HTTP 503 status code for SEO compliance.
- Whitelist options for specific IP addresses.
- Simple and clean design for the maintenance page.
- Localization support for multiple languages.
